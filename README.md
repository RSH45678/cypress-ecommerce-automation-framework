# Cypress E-Commerce Automation Framework

## Project Overview

This project demonstrates end-to-end UI automation of an e-commerce application using Cypress and the Page Object Model (POM) design pattern.

## Features Automated

* Home Page Validation
* Product Selection
* Add Product to Cart
* Cart Validation
* Place Order
* Purchase Confirmation
* Data-Driven Testing using Fixtures

## Tech Stack

* Cypress
* JavaScript
* Page Object Model (POM)
* Fixtures
* Git & GitHub

## Project Structure

* Page Objects
* Fixtures
* Test Scripts
* Support Files

## Run Project

npm install

npx cypress open

npx cypress run
